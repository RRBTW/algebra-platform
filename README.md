# Платформа для тестирования по алгебре — 8 класс (Макарычев)



Локальная веб-платформа для контрольных работ. Учитель запускает сервер на своём ПК, ученики подключаются по WiFi.

---

## Быстрый старт

### 1. Установите Node.js
Скачайте с https://nodejs.org/ (версия LTS), установите.

### 2. Запустите платформу
Дважды кликните **`start.bat`** — он установит зависимости и запустит сервер.

Либо вручную в терминале:
```
cd algebra-platform
npm install
npm start
```

### 3. Подключение
В консоли появится:
```
🟢 Сервер запущен!

   Ученик:  http://192.168.X.X:3000/student/
   Учитель: http://192.168.X.X:3000/admin/
```

- **Учитель** открывает `http://192.168.X.X:3000/admin/` (пароль: `admin123`)
- **Ученик** открывает ссылку `/student/` на своём телефоне/ноутбуке (должен быть в той же WiFi сети)

```

├── server.js          — Express + Socket.IO сервер
├── db.js              — SQLite база данных

│   │   ├── tests.html     Список тестов

│   │   └── results.html   Результаты с разбором
│   ├── admin/         — Панель учителя
│   │   ├── index.html     Управление тестами (вкл/выкл)
│   │   ├── live.html      Реалтайм монитор
│   │   ├── results.html   Результаты всех учеников + аналитика по темам
│   │   └── editor.html    Редактор вопросов
│   └── shared/
│       └── style.css
└── algebra.db         — База данных (создаётся автоматически)
```

## Возможности

**Ученик:**
- Вход по имени, сессия сохраняется в браузере
- Все вопросы на одной странице (скролл)
- Ответ отправляется сразу при выборе (прогресс сохраняется)
- После завершения: результат, подсветка ошибок, объяснения

**Учитель:**
- Включать/выключать тесты
- Реалтайм монитор: видеть кто решает, прогресс, правильность ответов
- Таблица результатов с подробным разбором
- Анализ слабых тем (% правильных ответов по каждой теме)
- Редактор: добавлять/редактировать/удалять вопросы

## Настройки

Пароль администратора (по умолчанию `admin123`) можно изменить через переменную окружения:
```
set ADMIN_PASS=мойпароль
npm start
```

Порт (по умолчанию 3000):
```
set PORT=8080
npm start
```
